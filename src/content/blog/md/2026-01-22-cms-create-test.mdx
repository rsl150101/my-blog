이곳은 본문 영역입니다. `line-height`와 `word-break` 속성이 적용되어 한글 가독성이 어떻게 나오는지 확인해봅니다. 글자가 너무 빽빽하지 않고, 문단 간의 간격이 적절한지 체크해보세요.

## 중제목 (H2): 코드 블록 테스트

가장 중요한 코드 하이라이팅과 <span>**줄 번호 (Line Numbers)**</span>가 잘 나오는지 확인합니다. 왼쪽 여백(`padding-left`)이 충분해서 숫자가 짤리지 않아야 합니다.

```jsx
import React, { useState, useEffect } from "react";

// 코드 블록 테스트 컴포넌트
const CodeBlockTest = () => {
  const [count, setCount] = useState(0);

  useEffect(() => {
    console.log(`현재 카운트는 ${count}입니다.`);
  }, [count]);

  return (
    <div className="container">
      <h1>Hello, Gatsby!</h1>
      <button onClick={() => setCount((prev) => prev + 1)}>클릭하세요</button>
    </div>
  );
};

export default CodeBlockTest;
```
